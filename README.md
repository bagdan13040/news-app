# DuckDuckGo News Search CLI

Простая консольная утилита для поиска новостей: вводите запрос, получаете список превью, выбираете статью и читаете полный текст.

## Установка

```bash
python -m pip install -r requirements.txt
```

## Запуск

```bash
python news_parsing.py "инфляция"
```

По умолчанию выводится 6 последних превью. Чтобы изменить количество превью, добавьте параметр `--limit` или `-n`.

Пример:

```bash
python news_parsing.py "искусственный интеллект" --limit 8
```

## Как пользоваться

1. Введите поисковую фразу.
2. Получите список заголовков с кратким описанием и ссылками.
3. Введите номер интересующей новости, чтобы загрузить полный текст статьи.
4. После чтения можно выбрать другую статью или выйти, введя `q`.

## Особенности

- Использует DuckDuckGo HTML-выдачу, не требует API-ключа.
- Предварительный просмотр включает заголовок, описание и ссылку.
- Полный текст формируется из `<p>`-абзацев статьи (если доступно).
- Обработка ошибок: в случае проблем с загрузкой новости программа предлагает выбрать другую статью.

## Kiwi mobile API

Для мобильного приложения (например, Kiwi Browser/Android) доступно HTTP API:

- `GET /search?q=<запрос>&limit=<число>` — возвращает массив превью (title/snippet/link) для отображения в списке.
- `GET /article?url=<ссылка>&max_chars=<число>` — загружает и очищает текст статьи с заданным ограничением длины.

Сервер запускается так:

```bash
python kiwi_news_api.py
```

По умолчанию слушает на `http://0.0.0.0:8080`. Мобильное приложение может всюду выполнять запросы к этим эндпоинтам, получая JSON для отображения.

## KivyMD (Kiwi) standalone app

Если вы хотите полноценное приложение на KivyMD, запустите экран поиска с `kivy_news_app.py`:

```bash
python kivy_news_app.py
```

Приложение раскрывает тот же поток:

- поле для ввода запроса и кнопка «Искать».
- список карточек с заголовками и сниппетами (DuckDuckGo HTML).
- кнопка «Читать», которая показывает содержимое статьи в диалоговом окне.

Вы можете собрать этот интерфейс в нативное мобильное APK через `buildozer` или `p4a` и использовать в Kiwi Browser.
